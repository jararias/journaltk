# Journal's toolkit

### Installation

```python
python3 -m pip install git+https://github.com/jararias/journaltk@main
```

### Usage description

```journaltk --help```

```journaltk pdf-doi <pdf-file-name>```

```journaltk pdf-rename <pdf-file-name>``` or ```journaltk pdf-rename --doi <doi> <pdf-file-name>```

```journaltk bibtex <doi>```
